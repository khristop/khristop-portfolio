Personal Branding

Description

My personal branding is a project I started in 2023. I wanted to create a brand that represents me and my work in a simple, modern, and clean way. I also wanted to create a brand that is flexible and can be used in different applications.

Process

I started the process by creating thumbnail for a logo which represents me. I used the words: Mentorship, curiosity, details matter, constant learning, humble, problem solving, ready to help and efficient. After that, I created created more detailed sketches and receive feedback to procced with the next step. I created the logo in Adobe Illustrator and exported it to Figma to use it in my portfolio. The portfolio website was created following the same principles as the logo.
