Website redesign: Humble Bundle

Description

Humble Bundle is a digital distribution platform that offers a unique approach to selling video games, ebooks, software, and other digital content. The company was founded in 2010 and has gained popularity for its pay-what-you-want pricing model and its charitable contributions.

Website
Problem

The following problems were defined by using a user-centric perspective. Although, these affect directly the user experience in the website.

  • Navigation and Information Architecture
  • Contrast
  • Colour palette
  • Layout
  • Readability
Solutions

The following problems were defined by using a user-centric perspective. Although, these affect directly the user experience in the website.

  • Improve the navigation state, the user should always know where they are
  • Distinct groups of elements from other groups, mostly in cards.
  • Improve the call to actions by using contrast colours
  • Improved grids, cards and background
  • Avoid using a font family with thin serifs
Typography

The website must have typography that works well with dark backgrounds. So, I choose a set of modern and robust typographies to improve readability considering the aesthetics.

Colour Pallete

I kept the primary colours of the brand; however, I included the orange as an accent to enhance the contrast and focal point of the call to action.
